# LazyVim Config

Personal Neovim configuration based on [LazyVim](https://github.com/LazyVim/LazyVim).

## Install

Clone directly into your Neovim config directory:

```sh
git clone https://github.com/DarkFluid/lazy-vim.git "${XDG_CONFIG_HOME:-$HOME/.config}"/nvim
```

Then start Neovim:

```sh
nvim
```
